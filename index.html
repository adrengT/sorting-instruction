Setting up a Sorting List

  1. From the WordPress Dashboard, go to Products > Merchandising
  2. Once on the Re-order view, click on the + button at the end of the pre-made sorting list to make a new one.
  3. Once on the Create new sort view, click on the the Show Advance button on the top right. The Query Rules section will then expand.
  4. Under the Taxonomy heading, click on the Add Taxonomy button.
  5. Choose the taxonomy in the first field. In this case, you can choose Styles, Devices, OEMs, and Family. Leave the other options as they are.
  6. Once you've chosen, the box on the right will repopulate. Choose the taxonomy you want to sort. You can choose more than one but to sort categories in the front-end of the site, choose just one.
  7. Scroll down a bit until you get to the Interface section. Here, name the sorting list in the Title Field. I'm doing Style/Acessories for clarity later when we need to get back to it. The UI is a bit unintuitive right now so we'll have to name them properly to separate them from each other.
  8. In the Thumbnails section, Choose Yes so the sorting list displays the product images by default.
  9. In the Status section, uncheck all of the checkboxes by clicking on All two times, then choose Published. Only Published should be active here.
  10. Click on Create button to create the Sorting list.

Sorting Products

  1. After the Sorting list has been made, you can minimize the Sort List Setting by clicking on it.
  2. Drag and drop products up or down accordingly to reorder them.
  3. You can click on the Grid View icon on the upper right corner to display the products in a grid view instead.
  4. In the Automatic Order tab, we can let the tool automatically sort the products before we re-order them. This is useful when we only want a couple of products to be "pinned" and let everything else in the category be sorted by publication date (Date, in this case) for example.
  5. Click on the Update button to save the order.

Notes

For now, the plugin is unintuitive in the way it handles Sorting Lists. It does what we need it to do but we'll have to make one sorting list for every category/taxonomy we need to sort. In the future, I'll integrate this to the Taxonomy dashboard so we won't have to make one each time. Ideally, it will be auto-generated everytime we make a new taxonomy and we'll only have to sort them.
